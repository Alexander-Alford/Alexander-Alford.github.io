Game Engine

A 2D platforming game engine with custom assets. Built from scratch in C with the SDL2 library.

A 2D Game Engine

This project was one of my first major software development projects. The goal of the project was to improve my general programming skill and gain a better understanding of the C language.

Collision Detection and Resolution


One of the first parts of the engine developed was the collision detection and resolution system. Over time the system was revised several tiems to more accurately resolve collisions and to detect non-rectangular collisions.

Tiling and Graphics


Alongside the collision detection and resolution, a grid-based tiling system was developed to create the platforms in the game engine. These tiles were drawn with hardware accelerated rendering of select sections of .png images.

Object System


The object system allowed for the creation of objects that could interact with the platforms and eachother according to their AI or their controls. Objects can store data that is unique to each individual member as well as data that is shared among all of them.

Sounds and Timing


One of the last elements added was a sound system that would take a stack of queued sounds and play them in order, in addition to the background music. The timing system was also made to carefully control the game speed.
